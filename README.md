# Sistema Bancário em Python

Projeto desenvolvido para praticar os conceitos básicos de Python por meio da implementação de um sistema bancário simples utilizando funções.

## Funcionalidades

* Cadastro de usuários;
* Criação de contas bancárias;
* Depósito;
* Saque com validações de saldo e limite;
* Consulta de extrato.

## Tecnologias utilizadas

* Python 3

## Objetivo

O objetivo deste projeto é reforçar os conceitos de lógica de programação, funções, estruturas condicionais, estruturas de repetição, listas e dicionários, desenvolvendo uma aplicação executada no terminal para simular operações bancárias básicas.
